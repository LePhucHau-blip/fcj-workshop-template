## Tuần 1: Nền tảng AWS và Tổng quan dự án
**Thời gian: 17/04/2026 - 21/04/2026**

### Mục tiêu của tuần:
- Làm quen và có cái nhìn tổng quan về lộ trình học AWS Cloud Journey.
- Hiểu bài toán mà dự án **Smart Image Platform** hướng đến giải quyết.
- Xác định các dịch vụ AWS được sử dụng trong dự án.

### Kiến thức AWS:
- AWS Account, Region, Availability Zone.
- Kiến thức cơ bản về IAM.
- Tổng quan về Amazon S3, AWS Lambda, Amazon API Gateway và Amazon DynamoDB.
- Khái niệm về kiến trúc **Serverless**.
